💡 Как это работает?

1

Создание мероприятия

Администратор создает мероприятие с указанием названия и количества мест

2

Бронирование

Пользователи бронируют места, указывая свой Telegram ID и желаемое количество мест

3

Подтверждение

В течение установленного времени бронь должна быть оплачена, иначе она автоматически отменяется

🔌 API Endpoints

POST /events Создание мероприятия
GET /events Получение всех мероприятий
GET /events/{id} Получение конкретного мероприятия
POST /events/{id}/book Бронирование места
POST /events/{id}/confirm Подтверждение оплаты
